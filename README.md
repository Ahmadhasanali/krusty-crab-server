# Krusty Krab Server

API for Krusty Krab Server.

## prerequisites

1. Clone this Repo / Pull Main
2. install node modules

   ```js
    npm i
   ```

3. To start

    ```js
    npm start
   ```

   or

   ```js
    nodemon app.js
   ```

4. To use seeder use

   ```js
   npm seed
   ```

5. if nodemon not working, do this, and try again

   ```js
    npm i -g nodemon
   ```
